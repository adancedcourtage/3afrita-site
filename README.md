## Déploiement

Site statique. Le repo est prêt pour GitHub Pages.

### GitHub Pages

1. Aller dans `Settings` -> `Pages`.
2. Choisir `Build and deployment` -> `Source` -> `GitHub Actions`.
3. Vérifier que le domaine custom est `3afrita.com`.
4. Côté DNS, pointer le domaine vers GitHub Pages.
